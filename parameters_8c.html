Detailed Description

Implementation of parameter handling and configuration.

This file defines functions and data structures for managing parameters, including their values, endianness, scaling factors, and offsets. It also includes arrays for transmission (Tx) and reception (Rx) parameters.

float GetParameterValue ( _parameter parameter)

Retrieves the scaled value of a parameter.

This function calculates the value of a parameter based on its raw byte data, endianness, length, and scaling factor. It supports both Intel (little-endian) and Motorola (big-endian) formats.

Definition parameters.h:37

Array of reception (Rx) parameters.

This array contains configurations for parameters that will be received. Each element is of type _parameter, which includes:

  • dParams.factor: Scaling factor for the parameter.
  • dParams.length: Length of the parameter in bytes.
  • dParams.offset: Offset to be applied to the parameter value.
  • dParams.index: Index of the parameter in the message.
  • dParams.endian: Endianness of the parameter (INTEL or MOTOROLA).

int AnalogSensorRead ( _sensorInterface * sensorInterface)

Reads the analog value from a sensor.

This function selects the appropriate ADC channel, starts the ADC conversion, polls for the conversion to complete, and retrieves the ADC value. The raw ADC value is then converted to a percentage using the GetSensorValue function.

◆ GetSensorValue()

float GetSensorValue ( _sensorInterface * sensor)

Converts the raw sensor value to a percentage.

This function checks if the raw sensor value is within the defined range. If the value is out of range, it sets the fault status to SENSOR_STATUS_FAULT and returns an error value. Otherwise, it calculates the percentage of the sensor value within the range and sets the fault status to SENSOR_STATUS_OK.

Parameters
sensorPointer to the sensor interface structure.
Returns
float Returns the percentage value or -1 if the value is out of range.

int ReadAnalogSensorList ( _sensorInterface * sensors)

Reads a list of analog sensors.

This function iterates through the sensors array and reads each sensor using the AnalogSensorRead function. A small delay is added between readings to avoid overloading the system.

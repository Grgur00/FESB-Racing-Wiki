◆ canRxMessages

_canMessage canRxMessages[]
extern

Array of CAN messages for reception (Rx).

This array contains the configuration for CAN messages that will be received by the application. Each message is defined by the _canMessage structure, which includes:

  • id: The CAN message identifier.
  • dlc: The data length code (DLC) specifying the number of bytes in the message.
  • signalCount: The number of signals contained in the message.

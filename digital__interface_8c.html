Detailed Description

Implementation of digital interface configuration and control.

This file defines an array of digital interfaces, each representing a GPIO pin configuration. It also includes a function to set the state of a digital output pin.

Array of digital interfaces.

This array contains configurations for digital GPIO pins. Each element is of type _digitalInterface, which includes:

  • pinState: The current state of the GPIO pin (e.g., GPIO_PIN_RESET or GPIO_PIN_SET).
  • pin: The GPIO pin number.
  • gpioPort: The GPIO port to which the pin belongs.
  • timestamp: A timestamp field for tracking purposes (currently unused and initialized to 0).
